# Sample_SpotifyPlayer API React
Sample_SpotifyPlayer React/Mobx app.

Demo: https://jsdevpro.pl/spotify-react/

## Installation:

## Use git clone.
```
git clone https://github.com/jaman7/Sample_SpotifyPlayer.git
```
## install
```
yarn install
```

## run command

```
gulp
```
